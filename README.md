# School Management System

## Features
- Authentication
- Student Management
- Teacher Management
- Course Management
- Section Management
- Attendance Management
- Exam Management
- Result Management
- Role Based Access Control

## Technologies
Frontend:
- React
- Vite
- Bootstrap
- React Query

Backend:
- Node.js
- Express.js
- MySQL
- JWT Authentication

## Installation

### Backend

npm install
npm run dev

### Frontend

npm install
npm run dev
